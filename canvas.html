About the Project

This project explores handwritten digit recognition using TensorFlow.js. You'll interact with a canvas where you can draw a digit. The system then utilizes a pre-trained neural network, informed by the MNIST dataset, to predict the digit you've drawn. The network architecture consists of an input layer with 128 neurons, followed by two hidden layers with 20 and 10 neurons respectively. Finally, a 10-neuron output layer generates the predicted digit.
